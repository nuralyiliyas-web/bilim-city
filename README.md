# Bilim City

Интерактивный MVP учебной игры для 9 класса: район «Энергополис» объясняет основы электричества через короткие уроки и строительство города.

## Что реализовано

- экран входа без реальной регистрации;
- адаптивная 2D-карта Энергополиса;
- пять тем по физике с тремя вопросами в каждой;
- последовательное открытие уроков и зданий;
- прогресс и имя ученика сохраняются в браузере через `localStorage`.

## Запуск

1. Установите Node.js 20 или новее.
2. Самый простой запуск в Windows: дважды нажмите `start-bilim-city.bat`. Не закрывайте открывшееся черное окно: пока оно открыто, сайт доступен по `http://localhost:3000`.
3. Или в терминале VS Code в папке проекта выполните:

```bash
npm install
npm run dev
```

4. Откройте `http://localhost:3000`.

## Проверка production-сборки

```bash
npm run build
npm run start
```
